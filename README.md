# 🏨 Hotel Website

A modern and responsive hotel website designed to showcase accommodations, amenities, and services while providing visitors with an intuitive booking experience. The project focuses on clean design, responsiveness, and accessibility.

## ✨ Features

* Fully responsive design
* Hero section
* Room showcase
* Amenities section
* Gallery
* Customer testimonials
* Contact page
* Google Maps integration
* Smooth animations
* Mobile-friendly navigation

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Git
* GitHub
